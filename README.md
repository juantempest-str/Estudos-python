# Fundamentos de Análise e Desenvolvimento de Sistemas

Este repositório contém scripts e aplicações desenvolvidas para consolidar os fundamentos da programação em Python, refletindo a base prática da graduação em Análise e Desenvolvimento de Sistemas (ADS), com uma visão orientada à segurança da informação.

## Estrutura do Repositório

### 01. Buscador API (Interface Gráfica)
Aplicação desktop construída com `Tkinter` que consome uma API REST externa para buscar e exibir dados dinâmicos de forma estruturada.

### 02. Registro de Corridas (Banco de Dados)
Sistema de persistência de dados utilizando `SQLite`. Realiza operações de CRUD (Create, Read) estruturadas para armazenar e consultar métricas de forma local, evitando exposição de dados sensíveis em texto claro.

---
*Stack Tecnológica: Python, SQLite, Tkinter, Requests.*
